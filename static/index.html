Today is October 4, 2013. Several months ago the aspiration for to learn about programming was planted in me. It started off with choosing the right programming language. At that point, I knew exactly why I wanted to learn programming: to one day be able to develop my on web application. I Googled, I consulted the vibrant communities on Stackoverflow and Quora, and at the end of the day they introduced me to Python.

For the next couple weeks, I scoured the Internet for learning materials on Python. I came across Codecademy, Learn Python The Hard Way, Coursera, and various tutorials contributed by the friendly Python community. Each day, with whatever free time during, or after, work, I'd be going through the basics of my new-found interest by following the instructions I found. Eventually I became adept enough to write my own Python scripts to automate some of the work I do at work, a non-computer related job. I was proud with the accomplishments. Yet, with all that progress, I am still nowhere close to my ultimate goal, which is to create a web app.

One day I came across a project by Jennifer Dewalt that was featured on HN. It was very inspirational, and at the same time it made me realize I will probably never able to reach my goal if I don't adopt the same JFDI methodology.

Today, on October 4, 2013, I registered this domain on NameCheap.com for a year, and deployed a static page on Heroku using Node.js. I am choosing to use Node because it allows me to use a single language, JavaScript, for both front-end and back-end development. I realize that it won't be an easy path, since there's much to learn and explore as it's less beginner-friendly than, say, more mature and "magical" frameworks like Django and Rails. However, I believe by taking babysteps from familiarizing myself with JavaScript via the more visual front-end, I will eventually have enough experience to dive into the more abstract back-end without losing myself.

Starting today, for the next 181 days (in tribute to Jennifer and as a continuation thereof), I will create a page featuring something I learned that day, regardless how simple or complex.

This is the end of Day 1
